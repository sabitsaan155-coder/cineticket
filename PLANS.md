# PLANS

## Phase 1 — MVP (weeks 1-2)
- Project structure and environment
- Base pages (home, about, team)
- Auth: registration/login/logout
- Core entities (catalog + rentals + orders)
- Basic admin setup

## Phase 2 — Expansion (weeks 3-5)
- Reach 10+ apps
- 12+ models, relations, slug, validators
- Search/filters/sort
- User profiles and roles
- API 5+ endpoints
- Seed/fixtures

## Phase 3 — Polish (week 6)
- 15+ UI pages
- 15 CSS effects and 15 JS scenarios
- Custom 404/500
- ORM optimizations (select_related/prefetch_related)
- Reports/statistics

## Phase 4 — Deployment checklist (week 7)
- DEBUG=False, ALLOWED_HOSTS, SECRET_KEY via env
- PostgreSQL, gunicorn, nginx
- collectstatic and media
- HTTPS + redirect
- Logs, monitoring, backup
- Domain live before June 1, 2026
